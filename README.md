# NetKeyboard

## Requirements

Both scripts require the [pynput](https://pypi.org/project/pynput/) module:

```bash
pip install pynput
```

Runs on port 8000

## Server

```bash
py server.py
```

## Client

Run with server ip address as the first argument:

```bash
py client.py 192.168.1.1
```

Replace `192.168.1.1` with the server address.
